## 三、功能详情

### 3.1 Toast 通知组件

**文件**：`src/components/Toast/index.tsx`

**功能**：
- 四种通知类型：success（绿色）、error（红色）、warning（黄色）、info（蓝色）
- 自动消失：默认 3 秒，可配置时长
- 手动关闭：点击关闭按钮
- 堆叠显示：支持多个通知同时显示
- 退出动画：平滑的过渡效果

**API**：
```typescript
addNotification({
  type: 'success' | 'error' | 'warning' | 'info',
  message: string,
  duration?: number  // 毫秒，0 表示不自动消失
})
```

### 3.2 设置面板界面

**文件**：`src/components/SettingsPanel/index.tsx`

**功能**：
- 外观设置：主题切换、主题色选择
- 搜索设置：默认搜索逻辑、搜索结果数量
- 文件监控设置：自动扫描间隔、显示通知、显示隐藏文件
- 恢复默认设置功能

**数据存储**：
- 使用 localStorage 存储设置
- 键名：`something_settings`

### 3.3 标签编辑/删除对话框

**文件**：`src/components/TagPanel/index.tsx`

**功能**：
- 添加自定义标签：名称、显示名称、颜色选择
- 编辑标签：修改标签信息
- 删除标签：带确认对话框
- 右键菜单：编辑/删除操作入口
- 17 种预设颜色选择

**系统标签**：只读，不可编辑/删除
**自定义标签**：完全可管理

### 3.4 文件拖拽上传

**文件**：`src/components/FileList/index.tsx`

**功能**：
- 拖放区域：拖放文件到任意位置
- 视觉反馈：拖放时显示覆盖层和提示
- 处理多个文件：批量处理
- 进度提示：上传中状态显示
- 结果通知：成功/失败数量统计

**注意事项**：
- 目前使用临时文件路径，需要后续集成 Tauri 文件系统 API
- 需要添加实际的文件路径获取逻辑

### 3.5 深色模式支持

**文件**：`src/App.tsx`, `tailwind.config.js`, `src/stores/useStore.ts`

**功能**：
- 主题切换：浅色/深色模式
- localStorage 持久化
- 系统偏好检测（首次加载）
- 主题切换按钮（太阳/月亮图标）
- CSS 变量定义完整

**配置**：
- Tailwind：`darkMode: 'class'`
- 主题键名：`something_theme`

## 四、设计变更

### 4.1 架构变更

| 变更项 | 原设计 | 新设计 | 原因 |
