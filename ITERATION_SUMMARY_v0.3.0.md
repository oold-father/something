## 三、任务拆分

### Task #1：搜索关键字标签化输入

**描述**：搜索框支持多个关键字输入，每输入一个关键字前端界面使用一个小标签包裹住这个关键字，可以点击关闭小按钮，删除这个关键字。

**技术要点**：
- 前端：SearchBar 组件改造
- 实现标签渲染和删除逻辑

**验收标准**：
- [ ] 输入关键字后按回车，关键字变成标签显示
- [ ] 可以添加多个关键字标签
- [ ] 点击标签的关闭按钮可以删除该关键字
- [ ] 搜索功能正常工作（使用所有标签关键字）

**状态**：✅ 已完成

---

### Task #2：搜索后保留关键字标签

**描述**：搜索后搜索框保留已搜索的关键字，显示为标签状态。

**技术要点**：
- 前端：SearchBar 组件 + useStore 状态管理
- 保存搜索关键字到 store

**验收标准**：
- [ ] 搜索完成后，搜索框显示已搜索的关键字标签
- [ ] 可以删除某些关键字后重新搜索
- [ ] 可以继续添加新的关键字后重新搜索

**状态**：✅ 已完成

---

### Task #3：禁用列表外右键，列表内显示菜单

**描述**：除列表外其他位置全部禁用右键功能。列表内点击右键，弹出功能列表。

**技术要点**：
- 前端：App.tsx + FileList 组件
- 全局禁用右键，列表内启用

**验收标准**：
- [ ] 在应用非列表区域右键无反应
- [ ] 在文件列表区域右键显示功能菜单
- [ ] 菜单位置显示在鼠标点击位置附近

**状态**：✅ 已完成

---

### Task #4：右键菜单"添加标签"功能

**描述**：右键功能列表功能一，"添加标签"，可以选择已有文件标签建立文件和标签的关系支持多选。支持手动输入标签名，标签不存在则自动创建标签。

**技术要点**：
- 前端：新增 AddTagModal 组件
- 后端：新增批量关联接口

**验收标准**：
- [ ] 右键菜单点击"添加标签"弹出弹窗
- [ ] 弹窗显示所有可用标签，支持多选
- [ ] 可以手动输入新标签名
- [ ] 点击保存后，文件与标签成功关联
- [ ] 新输入的标签不存在时自动创建

**状态**：✅ 已完成

---

### Task #5：监控文件夹点击打开

**描述**：监控文件夹列表支持点击文件夹即打开文件夹。

**技术要点**：
- 前端：WatchedDirectories 组件点击事件
- 后端：新增打开文件夹路径的 Tauri 命令
- 配置 opener 插件权限

**验收标准**：
- [ ] 点击监控文件夹列表中的文件夹
- [ ] 系统文件管理器打开该文件夹
- [ ] 权限配置正确

**状态**：✅ 已完成
